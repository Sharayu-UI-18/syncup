# SyncUp 🎓

SyncUp is a social networking platform designed for college students to connect, collaborate, and build communities based on their interests, skills, and academic domains.

The platform helps students discover peers, interact with like-minded people, and work together on projects, learning, and opportunities.

## Features

- 👤 User Registration and Login
- 🤝 Student Collaboration System
- 📚 Domain-based Communities
- 💬 Connect and Interact with Students
- 🔍 Discover Students with Similar Interests
- 📅 Activity and Event Updates
- 🌐 User Profiles
- 📢 Share Ideas and Discussions
- 🔒 Secure Authentication System

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- PHP

**Database**
- MySQL

## Purpose

SyncUp aims to create a college-focused networking environment where students can:

- Connect with peers
- Explore different domains
- Collaborate on projects
- Build meaningful academic and professional networks
